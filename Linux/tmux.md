# tmux
- terminal multiplexer
- 터미널 화면을 분할하고, 세션을 생성하고 attach/detach를 자유롭게 할 수 있는 터미널 멀티플렉서이다

## tmux 구성
- session
    - tmux 실행 단위. 여러개의 window로 구성
- window
    - 터미널 화면. 세션 내에서 탭처럼 사용
- pane
    - 하나의 window 내에서 화면을 분할하여 사용하는 단위
- status bar
    - 화면 아래 표시되는 상태 막대


## tmux 명령어
- prefix 키 <ctrl+b>를 누른 후 다음 명령키를 따로 눌러야 동작한다
